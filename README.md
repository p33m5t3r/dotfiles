    - `~/.virtualenvs/general` setup
- fonts
    - install hack nerd ligature fonts
- terminal

- nvim

- desktop

    - set custom i3bar
- gui
    - chrome, spotify
- devel
    - will put common devel libs here

### installation

1. `git clone https://github.com/p33m5t3r/dotfiles.git $HOME/.dotfiles`
2. `cd ~/.dotfiles`
3. `chmod +x install.sh`
4. `sudo ./install.sh`
